# Тензорная Архитектоника

Научно-исследовательский репозиторий Тензорной Архитектоники — авторской системы Александра Лацинника.

## Назначение

Репозиторий собирает первичные научные тексты, формализации и инженерные следствия Тензорной Архитектоники в едином цитируемом пространстве.

## Архитектура корпуса

- `01-declarations/` — декларации и исходные положения;
- `02-axioms/` — аксиомы и инварианты;
- `03-theory/` — теоретические работы;
- `04-mathematical-apparatus/` — формулы, операторы и геометрии;
- `05-engineering-applications/` — проверяемые цифровые и прикладные реализации;
- `catalog/` — библиографическая опись корпуса.

Репозиторий находится в стадии первичной научной систематизации. Размещение текста означает фиксацию версии работы, но не прекращение её развития.
