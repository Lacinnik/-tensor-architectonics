# Реестр авторских ключей ТзАр

Реестр содержит только открытые ключи и их SHA-256-отпечатки. Закрытые ключи, кодовые фразы и резервные копии в репозиторий не публикуются.

## Статус

Реестр создан и пока пуст. `TZAR-AUTHOR-KEY-001` будет добавлен после того, как Александр Лацинник:

1. создаст ключ локально в TZAR Conductance;
2. сохранит зашифрованную резервную копию;
3. экспортирует открытую регистрацию с доказательством владения закрытым ключом;
4. явно передаст регистрацию для публикации.

До регистрации продукт различает два результата:

- подпись криптографически верна;
- подпись верна и открытый ключ доверен репозиторием.
