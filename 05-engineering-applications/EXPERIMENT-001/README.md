# EXPERIMENT-001: Проводимость Осевания

## Цель

Проверить TZAR-THEOREM-001 на конечной цепи представлений канонической Декларации 001.

## Конструкт

- источник: [TZAR-DECLARATION-001 — «Осевание»](../../01-declarations/DECLARATION-001-OSEVANIE.md);
- проверяемая теорема: [TZAR-THEOREM-001 — «Транзитивная проводимость»](../../03-theory/THEOREM-001-TRANSITIVE-CONDUCTANCE.md);
- оператор `σ`: выделение идентификатора, автора, осевой формулировки, версии и статуса.

## Цепь

1. `euclid-linear.json` — линейная форма с упорядоченными секциями;
2. `projective-graph.json` — графовая форма отношений;
3. `supra-envelope.json` — форма инвариантного конверта;
4. `negative-control.json` — намеренная подмена слова «сущей» словом «наблюдаемой».

Названия геометрий обозначают инженерные модели представления и не объявляются полными математическими реализациями соответствующих геометрий.

## Критерий

Для положительной цепи:

`σ(F₀) = σ(F₁) = σ(F₂)`.

Для отрицательного контроля:

`σ(F₋) ≠ σ(F₀)`.

## Запуск

```bash
node verify.mjs
```

Успешный запуск завершается кодом `0`. Любая утрата положительного инварианта или пропуск отрицательного контроля завершается ненулевым кодом.
