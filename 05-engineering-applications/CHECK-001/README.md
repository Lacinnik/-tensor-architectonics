# TZAR-CHECK-001: первая операционная проверка

Дата исполнения: 17 июля 2026 года.

## Цель

Проверить опубликованный контур `TZAR Conductance 1.0` после стабилизации релиза.

## Проверено

1. живая версия продукта соответствует `1.0.0 stable`;
2. SHA-256-печать релизного паспорта совпадает;
3. SHA-256-печать авторского обзора совпадает;
4. авторский обзор содержит все 12 решений;
5. доказательство владения `TZAR-AUTHOR-KEY-001` криптографически действительно;
6. отпечаток ключа присутствует в доверенном реестре;
7. последний полный запуск CI успешен;
8. последнее развёртывание GitHub Pages успешно.

## Результат

**PASS** — все восемь проверок пройдены.

Паспорт исполнения: [`TZAR-CHECK-001.json`](TZAR-CHECK-001.json).

## Граница

Проверка подтверждает операционную целостность опубликованного продукта, его паспортов и контура доверия на момент исполнения. Она не заменяет содержательной проверки новых научных работ и не утверждает неизменность внешней инфраструктуры после указанного времени.
