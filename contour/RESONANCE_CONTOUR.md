# Резонансный контур ТзАр

Резонансный контур превращает репозиторий из архива файлов в проводящую научную среду.

## Ось

Операционное определение осевания:

> Осевание — приведение субъектной структуры в состояние, при котором супра становится сущей.

Ось не доказывает утверждения и не заменяет исследование. Она удерживает различение между источником, формализацией, проверкой и применением.

## Пять узлов

1. **Декларации** — исходные положения и границы предмета.
2. **Аксиомы** — принятые инварианты с явным статусом.
3. **Теория** — гипотезы, модели, теоремы и их связи.
4. **Математический аппарат** — операторы, геометрии, метрики и вычислимые формы.
5. **Инженерные применения** — реализации, эксперименты и наблюдаемые результаты.

## Проводимость

Каждый новый конструкт проходит путь:

`Источник → Различение → Формализация → Проверка → Связь → Применение → Обратная связь`

Для оценки проводимости используются метрики ТзАр:

- `α` — соответствие;
- `IЯ` — интеграция ядра;
- `Cm` — контейнер смыслов;
- `Q` — качество отклика;
- `T` — текучесть.

Шкала: `0 / 0.25 / 0.5 / 0.75 / 1.0`.

## Геометрии

Конструкт может быть размечен одной или несколькими геометриями:

- Евклид;
- Лобачевский;
- Риман;
- проективная;
- супра.

Геометрия указывается только там, где она действительно меняет отношение элементов.

## Статусы

- `draft` — первичная фиксация;
- `candidate` — формализованный кандидат;
- `verified` — прошёл заявленную проверку;
- `canonical` — принят автором в канон;
- `rejected` — отклонён с сохранением основания.

Только Александр Лацинник как автор ТзАр переводит положение в статус `canonical`.
