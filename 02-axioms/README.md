# Аксиомы и инварианты

Раздел предназначен для положений, которым присвоен явный статус внутри Тензорной Архитектоники.

## Правило включения

Аксиома публикуется здесь только после:

1. точной формулировки;
2. определения области применимости;
3. указания связанных терминов и геометрий;
4. различения аксиомы, гипотезы и метафоры;
5. авторской фиксации статуса Александром Лацинником.

До такой фиксации положение размещается в теоретическом разделе со статусом `draft` или `candidate`.

Раздел намеренно не заполняется автоматически: отсутствие ложной аксиомы важнее полноты каталога.
