# Теорема 001: Транзитивная проводимость

## 1. Формулировка-кандидат

> **Если каждый переход в конечной последовательности геометрий сохраняет супровый инвариант конструкта, то супровый инвариант сохраняется между начальной и конечной формами этой последовательности.**

Теорема утверждает проводимость идентичности относительно супры через цепь допустимых преобразований. Она не утверждает тождество промежуточных или конечных проявленных форм.

## 2. Основание

Теорема выводится из канонической [TZAR-AXIOM-001 — «Супровый инвариант»](../02-axioms/AXIOM-001-SUPRA-INVARIANT.md).

Аксиома фиксирует сохранение супрового инварианта при одном допустимом переходе. Теорема распространяет это отношение на конечную последовательность таких переходов.

## 3. Формальная постановка

Пусть:

- `C` — конструкт;
- `G₀, G₁, …, Gₙ` — конечная последовательность геометрий проявления;
- `Fᵢ = F(C, Gᵢ)` — форма конструкта в геометрии `Gᵢ`;
- `Tᵢ,ᵢ₊₁` — допустимый переход от `Fᵢ` к `Fᵢ₊₁`;
- `σ` — оператор выделения супрового инварианта.

Для каждого `i ∈ {0, …, n−1}` выполнено:

`σ(Fᵢ) = σ(Fᵢ₊₁)`.

Тогда:

`σ(F₀) = σ(Fₙ)`.

## 4. Доказательство-кандидат

Для одного перехода утверждение совпадает с Аксиомой 001:

`σ(F₀) = σ(F₁)`.

Предположим, что для последовательности длины `k` выполнено:

`σ(F₀) = σ(Fₖ)`.

По условию допустимости следующего перехода:

`σ(Fₖ) = σ(Fₖ₊₁)`.

По транзитивности равенства:

`σ(F₀) = σ(Fₖ₊₁)`.

Следовательно, по математической индукции утверждение выполняется для любой конечной последовательности допустимых переходов.

## 5. Что именно сохраняется

Сохраняется не форма и не локальная метрика, а супровый инвариант конструкта.

Поэтому в общем случае допустимо:

- `F₀ ≠ F₁ ≠ … ≠ Fₙ`;
- изменение расстояний, кривизны, перспективы и способа представления;
- различие наблюдаемых свойств формы.

Недопустимо без отдельного основания утверждать, что сохранение супрового инварианта означает полную обратимость преобразований.

## 6. Следствия-кандидаты

1. Конструкт может сохранять идентичность при множественных изменениях представления.
2. Проводимость всей цепи зависит от проводимости каждого её перехода.
3. Один разрыв супрового инварианта лишает оснований утверждение о проводимости цепи в целом.
4. Историю преобразований можно использовать как проверяемое свидетельство непрерывности конструкта.
5. Конечная форма может быть соотнесена с исходной без требования их геометрического совпадения.

## 7. Инженерная интерпретация

Для цифрового репозитория теорема задаёт следующий принцип:

- научный конструкт получает устойчивый идентификатор;
- каждая редакция или формат является отдельной проявленной формой;
- переход фиксирует источник, преобразование и результат;
- цепь считается проводящей, пока на каждом шаге восстанавливается инвариант;
- изменение Markdown, DOCX, PDF, кода или интерфейса само по себе не уничтожает идентичность конструкта.

Это позволяет различать развитие работы и подмену работы.

## 8. Условия проверки

Инженерный эксперимент должен включать:

1. исходный конструкт с идентификатором;
2. не менее двух последовательных переходов;
3. разные геометрии или формы представления;
4. явный критерий выделения `σ`;
5. протокол происхождения каждого перехода;
6. проверку начальной и конечной форм;
7. намеренно внесённый разрыв как отрицательный контроль.

## 9. Границы

Теорема не утверждает:

- проводимость бесконечной последовательности;
- автоматическую допустимость композиции переходов;
- сохранение всех свойств конструкта;
- единственность оператора `σ`;
- физическую природу супры;
- возможность восстановить утраченную форму только по инварианту.
